# Design

- `DESIGN.md`: style spec gốc AI Labs.
- `workflow.md`: workflow thiết kế, quy ước vận hành, checklist Pencil.
- `system.pen`: design library gốc cho shell, tokens, primitives, patterns.
- `system.pen`: showcase canvas cho review, không phải file ref từ screen.
- `screens/*.pen`: screen family theo surface.
- `components/*.lib.pen`: component library dùng lại.
- `notes.md`: notes, tradeoffs, user flow, brainstorm.
