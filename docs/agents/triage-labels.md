# Triage Labels

Skills nói theo 5 role triage chuẩn. File này map role đó sang label thật đang dùng trong issue tracker của repo.

| Label trong mattpocock/skills | Label trong tracker này | Ý nghĩa |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer cần đánh giá issue |
| `needs-info` | `needs-info` | Đang chờ reporter bổ sung thông tin |
| `ready-for-agent` | `ready-for-agent` | Đã đủ spec, sẵn cho agent AFK |
| `ready-for-human` | `ready-for-human` | Cần human implement |
| `wontfix` | `wontfix` | Sẽ không xử lý |

Khi skill nhắc một role, dùng đúng label string ở cột bên phải.

Đổi cột bên phải nếu repo này dùng vocabulary khác.
